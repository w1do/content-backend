# Skill: Design Prototype (NeuralFlow AI/RAG)

## Overview
Этот навык предназначен для быстрого прототипирования интерактивных интерфейсов для ИИ-сервисов и RAG-систем. Он базируется на дизайн-системе ManuFlex, адаптированной под технологический стек искусственного интеллекта.

## Core Concepts
- **Transparency (Прозрачность):** Визуализация внутренних процессов ИИ (Chain of Thought, Retrieval).
- **Industrial Precision:** Использование точных сеток, глассморфизма и монументальной типографики.
- **Interactivity:** Анимированные переходы, "живые" 3D-объекты и динамические метрики.

## Folder Structure
- `reference/COMPONENTS.md`: Детальное описание 12 ключевых компонентов.
- `layouts/`: Примеры компоновки страниц (фронтенд и админ-панель).

## How to Use
1. **Identify the Process:** Определите, какой этап RAG-пайплайна вы хотите визуализировать.
2. **Select Components:** Выберите подходящие компоненты из `COMPONENTS.md` (например, `<Features />` для общего обзора или `<ThinkingTrace />` для детализации ответа).
3. **Apply NeuralFlow Style:** Используйте цвета `Neural Gold` (#C8A645) и `Logic Blue` (#4A90E2) для акцентов.
4. **Implement Glassmorphism:** Все наложения должны использовать эффект `Dark Surface Glass` с `backdrop-blur`.

## Verification Checklist
- [ ] Использованы шрифты `DM Serif Display` для заголовков и `DM Sans` для данных.
- [ ] Все вложенные блоки имеют скругления от 10px до 20px.
- [ ] Анимации "Lock-in" применены для появления ключевых элементов.
- [ ] Присутствуют визуальные доказательства надежности ИИ (источники, метрики точности).
