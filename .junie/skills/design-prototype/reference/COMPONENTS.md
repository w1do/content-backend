# Component Reference — NeuralFlow Design Prototype

В данном справочнике приведены детальные спецификации 12 ключевых компонентов для создания интерактивных прототипов ИИ-сервисов.

---

### 1. `<Header />` — Neural Hub
- **Назначение:** Навигация и системный мониторинг.
- **Визуальная метафора:** Командный мостик корабля.
- **Детали дизайна:**
    - Контейнер: `backdrop-blur-xl bg-white/5 border-b border-white/10`.
    - Справа: Виджет "AI Status" — плавающая панель с микро-графиком нагрузки на API.
    - Логотип: `font-serif text-2xl tracking-tighter`.

### 2. `<Hero />` — Intelligence Core
- **Назначение:** Первое впечатление и демонстрация мощи.
- **Визуальная метафора:** Ядро реактора или центр управления данными.
- **Детали дизайна:**
    - Фон: `bg-[radial-gradient(circle_at_center,_var(--color-primary-blue-dark)_0%,_transparent_70%)]`.
    - Текст: Заголовки `text-7xl lg:text-9xl` с анимацией `reveal-up`.
    - 3D-центр: Интерактивная сфера (Canvas/Three.js), имитирующая векторное пространство.

### 3. `<Features />` — RAG Pipeline Map
- **Назначение:** Образовательная визуализация процесса.
- **Визуальная метафора:** Производственный конвейер.
- **Детали дизайна:**
    - Структура: Линейная последовательность вложенных карточек.
    - Соединители: Анимированные линии (SVG stroke-dasharray) со световыми импульсами.
    - Индикаторы: Значки "Processing" на каждом активном этапе.

### 4. `<ProcessVisualizer />` — Data X-Ray
- **Назначение:** Глубокое погружение в технический процесс.
- **Визуальная метафора:** Рентгеновский снимок данных.
- **Детали дизайна:**
    - Сетка: Изометрическая проекция блоков памяти.
    - Анимация: "Стриминг" символов (Matrix style, но в сине-золотых тонах) при переходе из текста в вектор.

### 5. `<KnowledgeVault />` — Infinite Library
- **Назначение:** Показ масштаба базы знаний.
- **Визуальная метафора:** Монументальное хранилище или соты с данными.
- **Детали дизайна:**
    - Карточки: `rounded-[20px] bg-surface-dark border border-white/5 p-6 hover:border-accent/50`.
    - Сортировка: Интерактивные фильтры по типам данных (PDF, SQL, Web) с иконками `Lucide`.

### 6. `<Benifits />` — Agent DNA Showcase
- **Назначение:** Демонстрация специализаций ИИ.
- **Визуальная метафора:** Выбор персонажа в высокотехнологичной игре.
- **Детали дизайна:**
    - Панели: Вложенные блоки `bg-black/20` с вертикальными барами характеристик.
    - Описание: Текст `italic font-serif` для описания "личности" ассистента.

### 7. `<ThinkingTrace />` — Cognition Log
- **Назначение:** Объяснение логики ИИ.
- **Визуальная метафора:** Бегущая строка мыслей или лог-файл.
- **Детали дизайна:**
    - Стиль: Глассморфический контейнер с моноширинным шрифтом.
    - Иерархия: Вложенные пункты (Step 1 -> Step 2), которые раскрываются по мере "раздумий".

### 8. `<LiveChatDemo />` — Interactive Sandbox
- **Назначение:** Практическое тестирование.
- **Визуальная метафора:** Лабораторный стенд.
- **Детали дизайна:**
    - Чат-пузыри: `rounded-2xl` с градиентной обводкой для бота и простым фоном для пользователя.
    - Сноски: Маленькие "чипы" под ответом, ведущие к источникам в `KnowledgeVault`.

### 9. `<TrustMetrics />` — Hallucination Shield
- **Назначение:** Подтверждение безопасности.
- **Визуальная метафора:** Приборная панель самолета или медицинский монитор.
- **Детали дизайна:**
    - Графики: Радиальные индикаторы `stroke-dasharray` с неоновым свечением.
    - Текст: Метки "Verified by RAG" с иконкой щита.

### 10. `<Integrations />` — Ecosystem Hub
- **Назначение:** Показ связей.
- **Визуальная метафора:** Нейронная сеть или паутина соединений.
- **Детали дизайна:**
    - Центр: Главный узел "NeuralFlow".
    - Периферия: Парящие иконки сторонних сервисов с пульсирующими линиями связи.

### 11. `<Faqs />` — Industrial Accordion
- **Назначение:** Информационная поддержка.
- **Визуальная метафора:** Технический справочник или папка с чертежами.
- **Детали дизайна:**
    - Переходы: Плавное расширение высоты с `ease-in-out`.
    - Акценты: Золотая вертикальная линия слева от активного вопроса.

### 12. `<Footer />` — Global Anchor
- **Назначение:** Завершение пути и футер-навигация.
- **Визуальная метафора:** Фундамент здания.
- **Детали дизайна:**
    - Фон: `bg-gradient-to-t from-black to-[#0C1019]`.
    - Вордмарк: Массивный заголовок "NEURAL" с прозрачностью 3%, создающий эффект текстуры.
