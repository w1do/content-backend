# Документация Товаров

## Обзор
Система управления товарами позволяет хранить информацию о продукции, её характеристиках и принадлежности к категориям.

## API Эндпоинты

### Основные операции
- `GET /api/v1/products` - Получить список всех товаров. Возвращает массив товаров с базовой информацией.
- `GET /api/v1/products/{id}` - Получить детальную информацию о товаре по его ID.

## Структура данных (Product)
| Поле | Тип | Описание |
| :--- | :--- | :--- |
| `id` | Integer | Уникальный идентификатор |
| `category_id` | Integer | ID связанной категории |
| `name` | String | Наименование товара |
| `slug` | String | URL-псевдоним |
| `description` | Text | Описание товара |
| `attributes` | JSON | Дополнительные атрибуты товара (динамические характеристики) |

## Интеграция с категориями
Товары связаны с категориями. При запросе товаров через эндпоинт категорий (`/api/v1/categories/{id}/products`), можно получить товары не только конкретной категории, но и всех её дочерних категорий.

## Администрирование
Управление товарами осуществляется через Filament админ-панель. В интерфейсе доступны:
- Создание и редактирование товаров.
- Выбор категории.
- Редактирование текстового описания.
- Управление динамическими атрибутами (если настроено в Filament ресурсе).

## Техническая реализация
- **Entity**: `App\Domain\Entities\Product`
- **Repository**: `App\Domain\Repositories\ProductRepositoryInterface`
- **Handler**: `App\Application\Handlers\Products\GetProductsHandler`, `App\Application\Handlers\Products\GetProductByIdHandler`
- **Resource**: `App\Http\Resources\ProductResource`
