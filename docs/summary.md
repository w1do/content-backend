# Краткое описание реализованных изменений

## Категории и Иерархия товаров
- **CRUD**: Реализован полный цикл операций для категорий (создание, чтение, обновление, удаление).
- **Nested Set**: Использование иерархической структуры для категорий, что позволяет строить вложенность любого уровня (например, `оборудование -> газовое оборудование -> баллоны`).
- **Хлебные крошки**: Добавлен эндпоинт для получения цепочки родительских категорий для любой выбранной категории.
- **Интеграция с товарами**: Реализована фильтрация товаров по категории с автоматическим включением товаров из всех дочерних категорий.

## Архитектура и Качество кода
- **DDD & CQRS**: Код разделен на слои (Domain, Application, Infrastructure, Presentation). Все действия с категориями вынесены в Commands/Queries и Handlers.
- **Тестирование**: Весь новый функционал покрыт тестами (Pest). Добавлены тесты на CRUD, дерево категорий, хлебные крошки и вложенные товары.
- **Стандарты**: Код отформатирован согласно правилам проекта с помощью Laravel Pint.
- **Документация**: Обновлена Swagger (OpenAPI) документация и созданы файлы Markdown с описанием функционала.
