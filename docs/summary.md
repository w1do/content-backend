# Краткое описание реализованных изменений

## Категории и Иерархия товаров
- **CRUD**: Реализован полный цикл операций для категорий (создание, чтение, обновление, удаление).
- **Nested Set**: Использование иерархической структуры для категорий (пакет `kalnoy/nestedset`), позволяющее строить вложенность любого уровня.
- **Хлебные крошки**: Эндпоинт для получения цепочки родительских категорий.

## Товары (Products)
- **Каталог**: Реализован вывод списка товаров и детальной информации.
- **Связь с категориями**: Возможность получения товаров из всей ветки категорий (включая дочерние).
- **Гибкость**: Использование JSON-поля для динамических атрибутов товара.

## Контентная система (CMS)
- **Многофункциональность**: Поддержка различных типов контента (Блог, Страницы, Системные тексты).
- **API**: Отдельные эндпоинты для каждого типа контента.
- **Управление**: Полноценная админка на Filament для редактирования текстов.

## Архитектура и Качество кода
- **DDD & CQRS**: Четкое разделение на слои (Domain, Application, Infrastructure, Presentation). Использование команд/запросов и хендлеров.
- **Filament PHP**: Современная админ-панель для управления всеми сущностями проекта.
- **Тестирование**: Высокое покрытие тестами (Pest) для API и админки.
- **API Documentation**: Интеграция Swagger (L5-Swagger / OpenApi) для автодокументирования эндпоинтов.
